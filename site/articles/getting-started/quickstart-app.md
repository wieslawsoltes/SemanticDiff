# Quickstart: App

1. Start the desktop app.
2. Click **Open** and choose a Git repository folder.
3. Use the **Git** tab to select a local branch, remote branch, GitHub PR, or GitLab MR.
4. Use **Layout** and **Grouping** in the workspace toolbar to choose the graph shape.
5. Select a file node to synchronize the Files, Review, and Symbols panels.
6. Right-click nodes or file tree entries to open file diff tabs, blame tabs, symbol maps, or semantic maps.

## Recommended first review path

- Start with the default Diff Graph tab.
- Use Files search to focus the changed area.
- Open the Review tab to inspect linked comments.
- Open a file tab for detailed line-level reading.
- Open a Symbol Map when the change crosses several types or generated files.

## Useful shortcuts

- `Ctrl+O` opens a repository.
- `Ctrl+R` reloads.
- `Ctrl+L` recalculates layout and fits the graph.
- `F7` and `F8` navigate changes.
- `Ctrl+B` focuses Git reference search.
- `Ctrl+Shift+Y` focuses review search.
- `Ctrl+Shift+F` focuses symbol search.
