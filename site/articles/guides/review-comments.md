# Review Comments

## Review panel

The Review panel shows searchable GitHub/GitLab comments and threads. Selecting a thread focuses the related file node and line when location data is available.

## Comment annotations

Diff graph and file tabs can render comment annotations. Use annotation toggles to hide them when you want a clean code-only view.

## Interactions

- Hover an annotation to see a tooltip with comment metadata.
- Click an annotation to select the matching Review item.
- Use Review actions to open, reload, comment, reply, resolve, or reopen where provider support exists.

## Provider notes

GitHub review thread resolve/reopen requires authenticated GraphQL thread data. GitLab resolve/reopen uses resolvable MR discussions.
