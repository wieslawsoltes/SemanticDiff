# File Tabs

File tabs provide focused source reading without losing the graph context.

## Display modes

| Mode | Use |
| --- | --- |
| Diff only | Shows only changed hunks by default. |
| Changes | Keeps diff-only focused on change regions. |
| Full diff | Expands diff-only mode to full file context with diff markers. |
| Full file | Shows the file content with syntax coloring and diff annotations. |

## Viewer features

- syntax coloring through TextMateSharp,
- line number gutter,
- diff gutter and line annotations,
- fold-region metadata,
- selectable text,
- font-size dropdown,
- plus/minus font controls,
- `Ctrl`/`Cmd` + mouse wheel font zoom,
- context menu navigation to graph, map, file, symbol, and blame actions.
