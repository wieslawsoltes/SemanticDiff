# Guides

These guides focus on practical workflows inside the app and reusable packages.

## In this section

- [Git Workflows](git-workflows) covers branches, ranges, PRs, MRs, and workspace tabs.
- [Review Comments](review-comments) covers review thread interaction and annotations.
- [History and Blame](history-and-blame) covers commit timelines and file blame views.
- [File Tabs](file-tabs) covers diff-only/full-file rendering, annotations, folding, and font controls.
- [Symbols and Maps](symbols-and-maps) covers symbol panels, symbol graphs, and hybrid semantic maps.
- [Graph Export](exporting) covers SVG, PNG, and PDF export.
- [App Publishing](publishing) covers local and CI app publishing.
