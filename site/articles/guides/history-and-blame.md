# History and Blame

## History tabs

Open history from the Git tab for branches, PRs, or MRs. The timeline loads commits progressively while scrolling and draws lanes, commit dots, branch labels, and merge relationships.

## Commit actions

Right-click a commit to:

- copy the full hash,
- set it as the comparison range start,
- set it as the comparison range end.

## Blame tabs

Open blame from a file tree item, file tab, or graph node. The blame tab can show:

- commit cards grouped by blamed lines,
- an expanded file history timeline,
- a change graph with one diff node per loaded file-history commit.

Use the change graph when you want to see how file ownership and file content evolved together.
