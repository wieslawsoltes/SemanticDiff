# Graph Export

Use the workspace toolbar **Export** action to save the current graph as SVG, PNG, or PDF.

## Export behavior

- The export uses Skia-backed rendering data.
- The exported graph matches the current workspace layout.
- Visible annotations, grouping, node labels, and semantic edges are included.
- Exports are best created after using **Fit** or a deliberate viewport/layout command.

## Formats

| Format | Use |
| --- | --- |
| SVG | Vector sharing, documentation, scalable diagrams. |
| PNG | Screenshots, issue comments, quick previews. |
| PDF | Review packets and archival output. |
