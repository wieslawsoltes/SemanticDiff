# Symbols and Maps

## Symbols panel

The Symbols panel combines search, scope filters, kind facets, file facets, hot symbols, and result lists. Facets can be resized so large repositories remain navigable.

## Symbol graph

Open a symbol graph from a symbol row, hot symbol, file facet, or graph node. Symbol graphs show symbols and semantic edges only.

## Semantic map

Open a semantic map when you need both file context and symbol relationships. Semantic maps connect real diff file nodes to contained symbol nodes, then overlay filtered symbol relationships.

## Filtering

Symbol workspaces support filters for:

- text,
- scope,
- symbol kind,
- file,
- edge kind,
- graph view mode,
- layout,
- grouping.
