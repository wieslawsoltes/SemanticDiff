# SemanticDiff Documentation

SemanticDiff is both a desktop review application and a set of reusable libraries. These articles cover the app workflow first, then the extracted packages and integration points.

## Sections

- [Getting Started](getting-started) covers prerequisites, local builds, and first-run usage.
- [Packages](packages) maps each NuGet package to its responsibility and consumer scenarios.
- [Concepts](concepts) explains graph workspaces, semantic analysis, review workflows, and rendering.
- [Guides](guides) walks through day-to-day repository review tasks.
- [Reference](reference) documents CI, docs publishing, API coverage, dependency notes, and extraction planning.
- [API Reference](../api) contains generated .NET API documentation.
