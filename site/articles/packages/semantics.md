# SemanticDiff.Semantics

`SemanticDiff.Semantics` defines language-neutral symbol and relationship models.

## Responsibilities

- semantic navigation items,
- symbol graph nodes and edges,
- changed and linked symbol metadata,
- file-level fallback symbols,
- edge kinds such as reference, binding, resource, inheritance, generated-file, and containment,
- semantic-map filtering inputs.
