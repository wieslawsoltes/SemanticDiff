# SemanticDiff.Controls.Uno

`SemanticDiff.Controls.Uno` provides reusable Uno Platform controls for the app surfaces.

## Responsibilities

- graph canvas controls,
- rich file and diff text controls,
- annotation interaction controls,
- review, history, blame, and symbol graph surface primitives,
- Skia-backed export and rendering integration points.

The package targets `net10.0-desktop` with Uno and SkiaSharp dependencies.
