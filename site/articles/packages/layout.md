# SemanticDiff.Layout

`SemanticDiff.Layout` turns file, symbol, semantic-map, history, and blame structures into graph positions.

## Responsibilities

- layered, grid, compact-grid, status-lane, and automatic layout modes,
- grouping by folder, semantic structure, language, status, or no grouping,
- symbol-only graph layout,
- hybrid file + symbol semantic-map layout,
- blame change graph layout,
- workspace graph state reuse.
