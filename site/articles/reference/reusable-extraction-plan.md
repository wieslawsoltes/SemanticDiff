# Reusable Extraction Plan

The reusable package extraction work moved app-independent services and controls into standalone projects.

## Extracted areas

- core data contracts,
- diff parsing and file text rendering models,
- Git/reference/review/history/blame services,
- semantic graph and language provider services,
- layout and rendering services,
- host-independent workbench orchestration,
- Uno controls for graph, file, diff, and annotation surfaces.
